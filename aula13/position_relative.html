Exemplo de position: relative

O position: relative mantém o elemento no fluxo normal, mas permite deslocá-lo usando top, left, etc.

Resumo

  • ✔ Mantém o elemento no fluxo normal
  • ✔ Permite usar top, left, right, bottom
  • ✔ Mantém o espaço original
  • ✔ Serve de referência para elementos absolute
