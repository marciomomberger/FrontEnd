🎯 Transform com Explicação

1️⃣ Translate (Mover)

Hover

👉 transform: translateX(100px)
Move o elemento 100px para a direita (eixo X).

2️⃣ Scale (Aumentar)

Hover

👉 transform: scale(1.5)
Aumenta o tamanho em 50%.

3️⃣ Rotate (Girar)

Hover

👉 transform: rotate(45deg)
Gira o elemento em 45 graus.

4️⃣ Skew (Inclinar)

Hover

👉 transform: skew(20deg)
Inclina o elemento, criando efeito de distorção.

5️⃣ Combinação

Hover

👉 Combina várias transformações:
Mover + girar + aumentar.

6️⃣ Rotação 3D

Hover

👉 rotateY(180deg)
Gira o elemento no eixo Y (efeito 3D).

7️⃣ ScaleX

Hover

👉 scaleX(2)
Aumenta apenas na largura (horizontal).

8️⃣ TranslateY

Hover

👉 translateY(-50px)
Move o elemento para cima.
